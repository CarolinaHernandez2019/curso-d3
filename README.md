# Hub de aprendizaje D3 + React

Cuaderno personal de Carolina con los ejercicios del curso *D3 ❤️ React* de [React Graph Gallery](https://www.react-graph-gallery.com/).

Cada pestaña del sitio es un proyecto distinto: desde el primer gráfico de barras hasta una réplica del bubble plot de Gapminder.

## Cómo correrlo

```bash
npm install
npm run dev
```

## Stack

React 19, D3 7, Vite. Despliegue automático a GitHub Pages con la acción en `.github/workflows/deploy.yml`.
